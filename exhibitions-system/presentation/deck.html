  • ✓ دليل حسابات + قيود يومية تلقائية لكل عملية (بيع، شراء، تكلفة، ضريبة، عمولة).
  • ✓ تقارير: قائمة الدخل، الميزانية العمومية، ميزان المراجعة، دفتر الأستاذ، التدفق النقدي.
  • ✓ ذمم العملاء (البيع الآجل) والموردين، رواتب وعمولات وعهد.
